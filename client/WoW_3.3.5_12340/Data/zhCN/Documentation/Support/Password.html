密码问题

更改密码： 要更改密码，请访问魔兽世界站点并输入您的账号名和当前密码登录。登录后点击“更改密码”链接并输入您的新密码。密码更改后将立即生效。

获取丢失的密码： 如果您丢失或遗忘《魔兽世界：燃烧的远征》账号密码，请访问官方网站以获取更多信息。

如果在重新获得账号密码时遇到困难，请发送电子邮件至billing@battlenet.com.cn联系我们的客户服务团队。

密码安全： 我们推荐您采取下列措施以确保密码安全：

  • 确保您的密码不含任何字典中包含的单词（正序单词或倒序单词）。
  • 使用字母、数字和符号的组合。仅在密码后添加一串数字可能无法确保密码安全。
  • 请勿使用连续的键盘字母（例如qwerty）。
  • 请勿使用真实姓名、账号名、人物名及其变体。
  • 请勿使用重复而连续的字符（例如abcabc）。

更多信息： 如果您有其他任何密码管理问题，请访问魔兽世界站点。
