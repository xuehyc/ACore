连接和登录问题（PC）

请谨记：游戏服务器可能会因升级、维护和重启造成短期中断。但是，若遇到长期性连接问题，请向技术支持报告。

1. 登录错误： 如果您在尝试登录时发生连接故障，应访问我们的官方网站查看此次连接中断是否为预定服务器中断。您也可访问官方论坛查看其他用户是否遇到相同的问题。若数分钟后仍无法解决连接问题，您可联系技术支持。

2. 游戏安装完毕后无法安装补丁或安装补丁程序中断： 补丁服务器可能出现瘫痪或暂时超载情况。在发行大型补丁时容易出现这种状况。您可以尝试稍后安装补丁。

您也可尝试暂时关闭运行中的所有杀毒软件。下载时杀毒软件程序将扫描文件，这能造成下载延时甚至使下载流程中止。

3. 安装补丁后，游戏重启并试图再次安装补丁，如此循环： 这种情况通常是由病毒或电脑上安装的广告插件造成的。请检查系统是否含有病毒软件。

4. 密码丢失和账号维护： 《魔兽世界：燃烧的远征》提供重新获取密码服务。您可访问http://www.battlenet.com.cn

5. 延迟和滞后： 延迟可妨碍游戏流畅性，或是减小您的电脑与《魔兽世界：燃烧的远征》服务器的通信速度。通信缓慢将导致在游戏中执行某项命令与该命令在屏幕上生效两者之间产生“延时”。这可能与电脑执行大量运算时电脑运行缓慢的状况类似，但是在此种情况下，问题在于电脑性能，而游戏中的延迟则在于网络连接。

在大多数情况下，若速度缓慢状况是由延迟造成的，将在屏幕底部命令栏中的延迟指示条中显示延时。在指示条中将以三种颜色表示您的网络连接质量，您可将指针移至指示条查看准确的延时数据。就基本规则而言，延时少于100毫秒即表示您的网络连接状态良好（如果您使用的是拨号调制解调器，则延时数据可能会更高）。如果您的网络连接状态一直不好，可尝试联系技术支持以获取改善连接状态的更多信息。

6. 游戏中断： 出错和网络中断能造成突然的游戏中断。但是，如果您在执行某些交互操作（例如与商贩交谈或获取任务）时频繁发生游戏中断现象，您则可能遇到技术问题。这类中断通常与网络通信故障有关。请首先确保您的网卡具备最新驱动。您可尝试暂时关闭防火墙。也可尝试简化网络连接。若仍无济于事，请联系技术支持。
