视频问题（PC）

大多数视频错误是由于软件和设置造成的。若若列表中未包含您遇到的问题，或提供的步骤无法解决问题，请联系技术支持。

1. 游戏延时或不连贯：  提高游戏性能的主要方法是调整游戏图像设置。降低图像配置可增加游戏速度和流畅性，但画面质量也将随之下降。您可尝试多种设置，达到最佳画面质量与游戏速度之间的平衡。

游戏时可按下Esc键调出视频选项菜单调整设置。也可退出游戏通过Config.wtf文件调整设置。

  • 打开硬盘驱动器上的《魔兽世界》文件夹。
  • 打开WTF文件夹。
  • 在文本编辑器中打开Config.wtf文件。

这些设置允许高级用户定制游戏性能。要快速提高性能速度，可尝试或编辑下列某些指令。

    SET lod "1"
    将地形细节水平（LOD）设置为"1"或"0"。启用该功能将允许游戏表现较少的画面细节，进而提高性能。
    SET lodDist "100"
    设置地形细节水平（LOD）发生变化的距离。100是较为保守的设置，但是您可将该值降低至50以提高性能。只有当LOD设置为"1"时LodDist方可生效。
    SET doodadAnim "0"
    将该值设为"1"可启用环境动画，例如风车和飞鸟。游戏中的某些区域具有多个动画物体，因此将该值设为0能显著提高这些区域的游戏性能。
    SET mapShadows "0"
    关闭地形阴影显示（"1"为开启值）。这将稍微提高游戏速度。
    SET anisotropic "1"
    该值范围为1（关闭）至16。禁用该功能将显著提高不支持各向异性过滤的显卡的更新速度（帧/秒）。
    SET farclip "477.0"
    该值与视频选项菜单中的FarClip滑动条一致。该值范围为177至777。该设置对性能影响较小。
    SET pixelShaders "0"
    启用像素渲染可减少渲染周期，将显著提高支持像素渲染的显卡的性能。您也可尝试比较打开（"1"）或关闭（"0"）该设置时的游戏速度。在所有Macintosh版本中，该选项是禁用的。
    SET specular "0"
    只有当开启像素渲染时方可使用镜像选项。该选项打开特定地形纹理的镜像增强，给与某些物体闪亮和反射外观。启用镜像（"1"）对性能影响较小。
    SET fullAlpha "0"
    由于目前很多显卡不支持该功能，我们建议您将该选项设为禁用（"0"）。
    SET MaxLights "1"
    该值范围为1至4。许多显卡能很好地表现多重光源，但是对那些不支持多重光源的显卡而言，降低该值可显著提高游戏速度。
    SET gxVSync "0"
    该设置将禁用帧同步，允许跳帧现象。将该值设置为1可降低游戏速度，提高画面质量。
    SET frillDensity "1"
    该值范围为1（关闭）至128。该值越高，越能表现环境细节（草、岩石等）。性能将随着细节的增强而减低。

调整PC游戏性能的另一种方法是调整显卡的AGP Aperture（如果可用的话）。通常可将该值设置为显存容量的两倍（x2）。因此，若为64MB显卡，则将AGP Aperture设置为128。您可进一步调整该值。注意：该操作属高级操作。暴雪娱乐无法就设置显卡的AGP Aperture值提供指示。请联系您的电脑或声卡制造商获取帮助。

2. 在城镇中游戏时，游戏延时或不连贯：  若您在进入大型城镇时画面更新速度缓慢，尝试在"OpenGL"和"Direct 3D"模式之间切换渲染模式。

  • 打开硬盘驱动器上的《魔兽世界》文件夹。
  • 打开WTF文件夹。
  • 在文件编辑器中打开Config.wtf文件，例如记事本。
  • 选择SET gxApi "opengl"或SET gxApi "direct3d"。
  • 保存文件并重启《魔兽世界：燃烧的远征》。

3. 游戏材质闪光、闪烁或扭曲：  游戏中显示某些材质效果时，有的显卡可能会出现问题。图像问题可能包括：静态材质闪光、固体绿或白色斑驳材质、图形或材质缺失。

要解决这些问题，将镜面反射度和像素渲染设为"0"

  • 打开硬盘驱动器上的《魔兽世界》文件夹。
  • 打开WTF文件夹。
  • 在文件编辑器中打开Config.wtf文件，例如记事本。
  • 将SET specular "1"更改为SET specular "0"。
  • 将SET pixelShaders "1"更改为SET pixelShaders "0"。
  • 保存文件并重启《魔兽世界：燃烧的远征》。

4. 图形设置损坏或错误：  您可删除Config.wtf随时恢复游戏默认设置。该文件位于魔兽世界目录下的WTF文件夹中。下次运行游戏时将创建新的Config.wtf文件。退出游戏后将更新对游戏设置做出的最新更改。
