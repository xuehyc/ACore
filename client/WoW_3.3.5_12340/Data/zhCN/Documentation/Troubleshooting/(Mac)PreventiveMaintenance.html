预防性维护（Mac）

要防止游戏出错，最重要的步骤之一是确保您的系统处于最佳的游戏配置。

1. OS更新： 游戏在最新的、已打上补丁的操作系统中可达到最佳运行状态。对Macintosh用户而言，更新Mac OS X时可能会出现文件权限问题，导致游戏无法启动或使用某些游戏功能时异常退出。出现这种状况时，请按以下步骤修复《魔兽世界：燃烧的远征》所在的硬盘驱动器的文件权限设置。

    Mac OS X 10.3（Panther）：
  • 单击Dock栏中的搜索图标。
  • 打开应用程序文件夹。
  • 打开实用程序文件夹。
  • 双击磁盘工具应用程序图标。
  • 选择游戏所在的硬盘驱动器。
  • 选择磁盘修理工具选项。
  • 单击“修改文件权限”按钮。

2. OS设置： 您可更改电脑操作系统中的某些设置，实现更流畅的游戏性能并减少错误。

首先请确保已停用或关闭所有的电源管理设置。如果您有设置屏保，请在游戏之前将其设置为“无”。

请确保退出当前在Mac OS X下未使用所有应用程序。若Dock栏中的应用程序图标旁有一三角形标示，则表示该应用程序正在运行。

若系统被超频，可能会产生过多的热量，进而影响程序性能。若系统产生热量较高但并未超频，我们建议您添加风扇帮助散热。

3. 驱动程序更新： 确保硬件设备使用最新的驱动程序，这点至关重要。该类设备包括显卡和声卡。开始游戏之前请确保您的显卡和声卡具有最新的驱动程序。您应尝试访问您的电脑或显卡/声卡制造商站点，查看系统显卡/声卡对应的最新驱动程序。您可访问我们的站点查看主要制造商的联系信息。

苹果将为与苹果操作系统捆绑的ATI和NVIDIA视频硬件提供驱动更新。Mac OS X更新中涵盖了新的驱动程序。请使用系统参数应用程序中的软件更新面板，或访问Apple's Mac OS X 站点查看最新的Mac OS X更新。

虽然鼠标问题并不常见，但鼠标问题通常可通过更新软件即可得到解决。请确保您的设备使用的是制造商提供的最新驱动程序。某些定点设备同样利用专有软件，因此您可能需要更新这些设备。若仍存在问题，您可停用这类软件和/或尝试使用操作系统标准驱动运行游戏。

4. 驱动器维护： 若电脑上安装的Mac OS版本中包含某些未更新的“磁盘修复”软件，则可能会损坏游戏软件。如果您想重新安装游戏，可尝试停用所有的驱动器维护软件。
