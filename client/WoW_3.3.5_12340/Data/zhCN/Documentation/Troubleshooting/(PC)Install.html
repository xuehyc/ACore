安装问题（PC）

如果您在安装《魔兽世界：燃烧的远征》时发生任何错误，请检查您的DVD表面是否清洁且无划痕。即使极其微小的划痕或污垢都可能导致DVD-ROM驱动器无法阅读光盘。如果您的光盘表面清洁且无划痕，请尝试以下步骤看看是否能解决安装问题。

1. 关闭所有后台或杀毒程序：在某些电脑上，杀毒软件可能对安装程序造成干扰。请关闭所有杀毒软件并选择选择性启动看看是否能顺利完成安装。

2. 尝试在新的驱动器中运行光盘：如果您的电脑具有另一DVD-ROM驱动器，尝试在驱动器中运行光盘看看是否能顺利完成游戏安装。某些驱动器阅读受损光盘的能力比其他驱动器强。

3. 联系网易：若安装《魔兽世界：燃烧的远征》时仍然发生错误且无法完成安装，请联系技术支持获取更多指示信息。
