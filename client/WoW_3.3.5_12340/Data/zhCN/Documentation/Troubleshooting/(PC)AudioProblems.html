音频问题（PC）

以下列出的是某些最为常见的声音问题，如果其中未包含或无法解决您遇到的问题，请联系技术支持。

1. 音频声音忽高忽低： 这通常是由于音频设置不当造成的。您应删除Config.wtf文件使游戏恢复至初始设置。该文件位于魔兽世界WTF文件夹（/../WoW/WTF/Config.wtf）。

2. 声音持续减弱直至完全听不见： 尝试重新安装DirectX以及声卡的最新驱动。这可能是由于某些程序修改了DirectSound或设备设置。

3. 完全听不见声音：  打开Sound.log文件查看初始化错误。该文件位于《魔兽世界》根目录下。若存在错误，请将该文件以及问题描述提交给技术支持。

4. 游戏时声音间歇或爆音： 卸载所有可能在系统上安装了音频编解码器的程序（例如视频/音频播放器）。游戏期间关闭其他音频软件。例如MP3播放器，音频播放器将占用运行《魔兽世界：燃烧的远征》所需的系统资源，且通常会造成声音变调。

5. Audigy 2崩溃： 如果您使用的是Audigy 2并遇到游戏崩溃或系统重启现象，您可以尝试重新配置您的音频输出。打开\SBAudigy2\Program\WDM\COMMON\目录查找名为CtPanel.exe的文件，您可以利用这个工具修改设备设置。关闭“扬声器设置”中的CMSS 3D选项。

6. 与音频相关的性能问题： 如果Windows在运行游戏时遇到性能问题，您可以尝试暂时停用声卡设备。若此举生效，您可联系技术支持 提交如何提高与性能相关的系统性能的报告。

  • 右键单击“我的电脑”并选择“属性”。
  • Windows 2000用户应单击硬件选项卡。
  • 单击“设备管理器”。
  • 展开声音、视频和游戏控制器选项。
  • 选择声卡并单击属性按钮。
  • 在设备用法选项中选择停用该设备。
  • 单击“确定”按钮。
  • 重启电脑并再次进入游戏。
