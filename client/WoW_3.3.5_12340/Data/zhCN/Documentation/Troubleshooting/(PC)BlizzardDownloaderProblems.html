暴雪下载器问题：（PC）

如果您在运行暴雪下载器时发现传输速度缓慢，请首先保持耐心。该传输方式的特点是它首先必须要花一些时间来使得传输连接成功。如果10-15分钟后您仍然觉得下载速度非常之慢或者您未能与下载服务器连接，请尝试以下步骤。

1. 关闭后台应用程序： 您可访问关闭后台应用程序页面获取关闭所有不需要的进程的说明，然后尝试重新启动下载器。若数分钟后下载速度仍然非常缓慢，请进入下一步。

2. 防火墙： 如果您运行了防火墙，请确认它设置为允许TCP（也称TCP/IP）端口3724、6112以及6881-6999的入连接。如果想要进一步了解怎样在一些常见防火墙上开放端口，请参阅高级网络章节中的网络帮助页面。如果其中未列出您使用的防火墙或您需要更多帮助，请参阅您的防火墙相关文档或者联系该防火墙的制造商。

重要说明： 请记住，与个人预计相反，许多常见防火墙被禁用后会极其限制通信量。请确认您所安装的所有防火墙安装配置正确并且处于激活状态，这样才能够有效使用下载器。

3. 路由器：

  • 请确认路由器配置允许通过端口3724、6112和6881-6999只转递到您一个人的电脑上。通过同一端口转递到连接到此路由器上的多个计算机上会引起冲突，这会使得下载速度缓慢。
  • 请确认没有计算机处于DMZ（为不信任系统提供服务的孤立网络段）。如果计算机处于DMZ，就会产生冲突，也会导致下载速度缓慢。
  • 确认同一网络上的其他计算机没有运行任何需要大量带宽的程序。记住路由器仅仅使得所有的计算机能够共享连接。它不能增加连接带宽！

4. 广告软件/间谍软件/病毒扫描： 系统中的广告软件、间谍软件或病毒也可能导致相同的症状。请确信已用最新的病毒库全面扫描了您的系统。如果您需要了解更多的关于怎样获得病毒扫描器的信息，请访问我们的电脑安全页面。

5. 升级网卡驱动： 如果您的下载客户端还有问题，我们强烈建议您确认您的网卡是否已安装了最新的驱动。很多人表示升级网卡驱动解决了他们独特的问题。

6. 联系技术支持： 若仍存在问题，请联系技术支持寻找一个可行的解决方案。
