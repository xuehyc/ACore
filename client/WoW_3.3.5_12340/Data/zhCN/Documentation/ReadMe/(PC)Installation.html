安装（PC）

如果您的电脑上安装有《魔兽世界：燃烧的远征》Beta版，请在安装零售版之前卸载它。

要安装《魔兽世界：燃烧的远征》，请将安装DVD放入您的DVD-ROM驱动器。如果您的电脑启用了自动执行功能，将显示《魔兽世界：燃烧的远征》启动画面。从列表中选择“安装燃烧的远征”并根据屏幕上的说明进行操作。如果您的电脑未启用自动执行功能，打开桌面上的“我的电脑”图标或单击“开始”按钮在菜单中选择“我的电脑”。然后打开您的DVD-ROM驱动器对应的驱动器名，双击“安装”图标并根据屏幕上的说明进行操作。

* 请确保您的硬盘驱动器至少有12G空间。

注意：如果您在安装《魔兽世界：燃烧的远征》时遇到问题，请参阅安装问题疑难解答。
