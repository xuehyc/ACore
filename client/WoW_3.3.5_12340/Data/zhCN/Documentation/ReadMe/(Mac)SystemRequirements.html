系统需求（Mac）

以下列出的是《魔兽世界：燃烧的远征》系统最低配置需求。虽然某些用户在低于这些配置需求的电脑上也能运行游戏，但是暴雪娱乐技术支持团队将只为满足最低系统配置需求的用户提供帮助：

Mac OS X 10.4.11或更高版本

  • PowerPC G5 1.6 GHz 或Intel Core Duo处理器。
    （推荐：Intel 1.8 GHz 或更高性能的处理器）
  • 1GB RAM。
    （推荐：2GB RAM）
  • 拥有硬件T&L 功能的64MB显存3D显卡，如ATI Radeon 9600或NVIDIA GeForce Ti 4600或更高性能显卡。
    （推荐：拥有顶点及像素渲染功能的128MB显存3D显卡，如ATI Radeon X1600 或 NVIDIA 7600或更高性能显卡）
  • 12G或更多硬盘空间。
  • 网络连接：宽带。
  • 鼠标：推荐使用带滚轮的多按钮鼠标。

游戏需求

  • 《魔兽世界：燃烧的远征》是《魔兽世界》的资料片，为了能够安装和玩资料片，您必须要有一个正确安装的《魔兽世界》。
  • 你必须在玩《魔兽世界：燃烧的远征》前升级您的《魔兽世界》账号。
  • 您必须与宽带网络连接方可玩《魔兽世界：燃烧的远征》。
