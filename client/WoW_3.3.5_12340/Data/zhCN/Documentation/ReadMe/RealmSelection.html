选择服务器（PC/Mac）

《魔兽世界》中的每个人物都生活在某个独特的服务器中。所有的服务器都拥有相同的《魔兽世界》壮丽场景，但又各具特色。当您以新账号首次登录时将显示选择服务器屏幕，屏幕中将列出以下问题：

服务器类型： 标准服务器（PVE），玩家对玩家（PVP）服务器，角色扮演（RP）服务器或角色扮演玩家对玩家 （RPPVP）服务器。

  • 在标准服务器中（PVE），您可集中精力进行冒险、攻击怪物以及完成任务而无需担心来自对立阵营玩家的攻击。您也可在双方愿意的情况下去指定PVP区域攻击其他玩家，或是攻击对立阵营的NPC，但是所有的攻击操作均非强制性的，玩家可拒绝PVP。
  • 在玩家对玩家（PVP）服务器中，玩家可在特定区域直接体验与对立阵营的玩家惊心动魄的激战。在您的起始地点周围的少数区域，您仍处于被保护状态（跟标准服务器相同），但是玩家可在远离起始地点的敌对区域与对立阵营的玩家交战。（如果您是部落玩家，则联盟玩家是您的敌人，反之亦然。）

根据您做出的选择，我们将建议最适合您的人物参数的服务器。如果您不喜欢我们为您选择的服务器，也可选择不同的服务器主机。一旦您选定服务器并创建人物，就无法更改该特定人物所在的服务器，因此请仔细选择。（如果您想与服务器中的朋友一同游戏，我们建议您提前联系他们以确定他们所在的服务器。）尽管您的人物无法更改服务器，但是基于账号的服务器选择并无任何限制条款，因此请自由地在《魔兽世界》各个服务器中进行探险。
