系统需求（PC）

以下列出的是《魔兽世界：燃烧的远征》系统最低配置需求。虽然某些用户在低于这些配置需求的电脑上也能运行游戏，但是暴雪娱乐技术支持团队将只为满足最低系统配置需求的用户提供帮助：

Windows(R)系统/OS: 2000 (Service Pack 4)/XP (Service Pack 2)

  • Intel Pentium 4 1.3 GHz 或AMD Athlon XP 1500+处理器。
    （推荐：双核处理器，如Intel Pentium D或AMD Athlon 64 X2）
  • 512 MB内存，Vista用户需1GB内存。
    （推荐：1 GB 内存，Vista用户需2GB内存）
  • 拥有硬件T&L功能的32MB显存3D显卡，如 ATI Radeon 7200 或NVIDIA GeForce 2或更高性能显卡。
    （推荐：拥有顶点及像素着色功能的128MB显存3D显卡， 如 ATI Radeon X1600 或 NVIDIA GeForce 7600 或更高性能显卡）
  • 12G或更多硬盘空间。
  • DirectX(R) 9.0c或更高。
  • 声卡：DirectX兼容声卡。
  • 网络连接：宽带。
  • 鼠标：推荐使用带滚轮的多按钮鼠标。

游戏需求

  • 《魔兽世界：燃烧的远征》是《魔兽世界》的资料片，为了能够安装和玩资料片，您必须要有一个正确安装的《魔兽世界》。
  • 你必须在玩《魔兽世界：燃烧的远征》前升级您的《魔兽世界》账号。
  • 您必须与宽带网络连接方可玩《魔兽世界：燃烧的远征》。
