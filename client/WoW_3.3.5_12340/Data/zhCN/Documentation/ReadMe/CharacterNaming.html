人物命名（PC/Mac）

创建您的人物名称

当您完成人物的外貌设计之后，即可为人物命名。在您的人物所在的服务器中，该名称是唯一的。您的人物将在以后的游戏生涯中使用该名称，因此请仔细确认您所创建的名称。

请注意查阅我们的命名规则：您一定不愿意重新为人物命名或招致其他惩罚。您不能在人物名称中使用任何非法的、有害的、威胁性的、辱骂性的、令人生厌的、诽谤的、粗俗的、淫秽的、引起憎恨的、带种族歧视色彩的、或其他引起反对的字眼。在为人物/公会/宠物或任何定制内容命名时使用所有这些字眼均是不适当的。您可以访问 http://www.warcraftchina.com/ 查阅更详细的命名规则来获取更多信息。
